# **App Name**: AgroCast

## Core Features:

- AI-Based Climate Risk Forecast: Predict pest attack probability, crop disease outbreak, water shortage, and extreme weather risk using satellite and weather datasets. Show risk map for the next 7-90 days.
- Profit Prediction Engine: Suggest the most profitable crop combination based on land area, soil details, and budget. Provides expected ROI, cost breakdown, and historical market price trends using a tool.
- Agro-Match Marketplace: AI-powered matching of farmers with buyers, suppliers, storage units, and transport providers. Deals are executed automatically upon meeting price conditions.
- Blockchain-Backed Farm Digital Passport: Every harvest batch gets a unique blockchain ID storing fertilizer, pesticide, water usage, carbon footprint, crop variety, and location details.
- Risk Map Display: Display the climate risk forecast on a map, visualizing potential challenges for different regions.
- Data Input Interface: Intuitive user interface for farmers to input land details, soil information, and budget parameters.

## Style Guidelines:

- Primary color: Earthy green (#6B8E23) to evoke nature, growth, and agriculture.
- Background color: Light beige (#F5F5DC), a desaturated hue similar to the primary color, to offer a neutral backdrop that doesn't distract from the data visualizations.
- Accent color: Golden yellow (#FFD700) analogous to green, used to highlight key metrics, CTAs, and important data points.
- Body and headline font: 'PT Sans' for a modern and readable user interface.
- Use clear and relevant icons to represent different crops, weather conditions, and risks.
- Implement a clean, card-based layout to present complex data in an easily digestible format.
- Incorporate subtle transitions and animations to enhance user engagement and guide the user through the platform.
